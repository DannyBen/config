# set/array-index

Update scalar array items by index.

## Source Files

```yaml
title: yaml
methods: [GET, SET]
```

```toml
title = "toml"
methods = ["GET", "SET"]
```

## Commands

```shell
config set methods.1 POST
```

## Result Files

```yaml
title: yaml
methods: [GET, POST]
```

```toml
title = "toml"
methods = ["GET", "POST"]
```
