Remove values from a scalar array

Usage:
  config array delete KEY VALUE... [options]
  config array delete --help | -h

Aliases:
  del

Behavior:
  Does nothing when KEY is not set.
  Ignores values that are not present.
  Deletes KEY when no values remain.

Parameters:
  KEY
    Dot notation string describing the intended array key

  VALUE...
    Scalar values to remove from the array

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

  --dry, -n
    Print the updated config without modifying the file

  --diff, -d
    Print a unified diff without modifying the file

  --color, -c
    Colorize diff output

Examples:
  # Remove a value
  config array delete -f config.toml roots /tmp

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config array delete roots /tmp

  # Use the short command alias
  config array del roots /tmp /var/tmp
