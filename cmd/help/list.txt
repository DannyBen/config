Show config values

Usage:
  config list [KEY]
  config list --help | -h

Aliases:
  ls

Parameters:
  KEY
    Optional key or table path to list

Options:
  --file, -f PATH
    Path to the config file

  --color, -c
    Colorize keys and separators

Examples:
  config list -f path/to/config.toml
  config list -f path/to/config.toml database
  config list -f path/to/config.toml database.port
  config list --color
  config ls --color
  config list
