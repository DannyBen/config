Show config values

Usage:
  config list [KEY]
  config list --help | -h

Aliases:
  ls

Parameters:
  KEY
    Optional key or table path to list

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

  --keys, -k
    Print only keys

  --match, -m TEXT
    Print only keys that contain TEXT

  --color, -c
    Colorize keys and separators

Examples:
  # List all values
  config list -f config.toml

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config list database

  # List values under a key
  config list database.port --color

  # List only keys
  config list --keys

  # List keys containing text
  config list --match model
