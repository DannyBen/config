Replace a scalar array

Usage:
  config array set KEY VALUE... [options]
  config array set --help | -h

Parameters:
  KEY
    Dot notation string describing the intended array key

  VALUE...
    Scalar values to store in the array

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

  --dry, -n
    Print the updated config without modifying the file

  --diff, -d
    Print a unified diff without modifying the file

  --color, -c
    Colorize diff output

Examples:
  # Replace all values
  config array set -f config.toml roots "$HOME/.cache" /tmp

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config array set roots "$HOME/.cache" /tmp
