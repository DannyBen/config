Delete a config value

Usage:
  config unset KEY [options]
  config unset --help | -h

Parameters:
  KEY
    Dot notation string describing the intended config key

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

  --in COLLECTION
    Remove a field from a record in COLLECTION. Must be used with --on.

  --on FIELD:VALUE
    Select a record by FIELD:VALUE. May be repeated.

  --if VALUE
    Only unset when the current value matches VALUE

  --if-exists
    Do nothing when KEY is not set

  --dry, -n
    Print the updated config without modifying the file

  --diff, -d
    Print a unified diff without modifying the file

  --color, -c
    Colorize diff output

Examples:
  # Simple delete
  config unset -f config.toml server.port

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config unset server.port

  # Delete only when the current value matches
  config unset server.port --if 3000

  # Delete if the key exists
  config unset server.port --if-exists

  # Delete from a matching array record
  config unset port --in servers --on name:api
