Dump config data

Usage:
  config dump [KEY] [options]
  config dump --help | -h

Parameters:
  KEY
    Optional key or table path to dump

Options:
  --file, -f PATH
    Path to the config file

  --json
    Dump as JSON instead of YAML

Examples:
  config dump -f path/to/config.toml
  config dump -f path/to/config.toml database
  config dump -f path/to/config.toml database --json
  config dump -f path/to/config.yaml server.hosts
  config dump
