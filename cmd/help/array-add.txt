Add values to a scalar array

Usage:
  config array add KEY VALUE... [options]
  config array add --help | -h

Behavior:
  Creates the array when KEY is not set.
  Ignores values that are already present.

Parameters:
  KEY
    Dot notation string describing the intended array key

  VALUE...
    Scalar values to add to the array

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

  --dry, -n
    Print the updated config without modifying the file

  --diff, -d
    Print a unified diff without modifying the file

  --color, -c
    Colorize diff output

Examples:
  # Add values
  config array add -f config.toml roots /var/tmp "$HOME/.cache"

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config array add roots /var/tmp "$HOME/.cache"
