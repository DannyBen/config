Open the config file in an editor

Usage:
  config edit [options]
  config edit --help | -h

Options:
  --file, -f PATH
    Path to the config file (default: $CONFIG_FILE)

Environment:
  EDITOR
    Editor command to run. Defaults to vi.

Examples:
  # Open a file
  config edit -f config.toml

  # Use a default file for repeated commands
  export CONFIG_FILE=config.toml
  config edit
